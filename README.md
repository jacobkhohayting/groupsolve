# GroupSolve static website

This is a responsive, multi-page HTML/CSS/JavaScript website based on the supplied draft and logo.

## Open it

Open `index.html` in a browser. All page links work locally because the site uses relative paths.

## Pages

- `index.html` — Overview
- `team.html` — Team overview
- `volunteers.html` — Interactive volunteer biographies
- `join-us.html` — Volunteer roles and interest form
- `classes.html` — Classes overview
- `summer-classes.html` — Summer class catalog
- `fall-classes.html` — Fall class catalog
- `competitions.html` — Competitions overview
- `fall-competition.html` — Fall competition details
- `events.html` — Filterable event list
- `contact.html` — Contact form and FAQs

## Quick edits

- Site colors: edit the variables at the top of `css/styles.css`.
- Text and dates: edit the relevant HTML page.
- Logo: replace `assets/groupsolve-logo.png` with another PNG using the same filename.
- Contact address: search the project for `hello@groupsolve.org`.
- Forms: the included forms are visual demos. Connect them to Formspree, Google Forms, Netlify Forms, or your own backend before publishing.
- Google Fonts are loaded from the web. The site falls back to common system fonts when offline.

## Publishing

This folder can be deployed as-is to GitHub Pages, Netlify, Cloudflare Pages, or another static host.
